## Image Guidelines

- **Format**: JPG for photos, PNG for logos/graphics
- **Size**: Optimize images (use tools like TinyPNG)
- **Resolution**: At least 1200px width for hero images
- **Alt Text**: Always add descriptive alt text in the code

## How to Use Images in Next.js

```jsx
import Image from 'next/image';

<Image
  src="/images/hero-pool.jpg"
  alt="Crystal clear swimming pool"
  width={1920}
  height={1080}
  priority
/>
```
